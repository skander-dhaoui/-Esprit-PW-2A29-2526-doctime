Test Mot de passe oublié

Test 1: Email valide (afnengorai@gmail.com)

Après le clic, attendez-vous à une redirection et un message "Si cet email existe..."

Test 2: Email invalide

Vous devriez voir un message d'erreur "Email invalide"

Test 3: Email inexistant

Vous devriez voir le message "Si cet email existe..." (pour des raisons de sécurité, on n'indique pas si l'email existe)

Instructions pour vérifier:

  1. Cliquez sur "Tester mot de passe oublié"
  2. Vérifiez que vous êtes redirigé vers le formulaire de réinitialisation
  3. Vérifiez le message de succès: "Si cet email existe, vous recevrez un lien de réinitialisation."
  4. Attendez 10 secondes et vérifiez votre email afnengorai@gmail.com
  5. Cherchez dans vos spams/promotions si l'email n'apparaît pas dans la boîte de réception
  6. Cliquez sur le lien dans l'email pour vérifier qu'il fonctionne

Dépannage:

  • Si aucun message n'apparaît: Vérifiez les logs du serveur PHP
  • Si vous recevez une erreur: Notez-la exactement
  • Si aucun email n'arrive: Vérifiez les spams/promotions
  • Vérifiez que la configuration Gmail est correcte dans config/mail.php
