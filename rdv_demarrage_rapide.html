⚡ RDV Avancée - Démarrage Rapide

1️⃣

Aller aux Rendez-vous

Cliquez sur "Rendez-vous" dans le menu admin

→ Ouvrir liste RDV
2️⃣

Cliquer le Bouton "Voir"

Cliquez sur le bouton 👁️ "Voir" pour n'importe quel RDV

Le bouton est dans la colonne "Actions"

3️⃣

Page Détails S'ouvre

La page affiche tous les détails du RDV avec commentaires

Attendez quelques secondes le chargement

4️⃣

Scroller vers "Ajouter Commentaire"

Scrollez vers le bas de la page

Vous verrez l'éditeur Quill.js

5️⃣

Ajouter Emoji

Cliquez sur le bouton 😊 Emoji

Choisissez un emoji dans la liste

6️⃣

Ajouter Image (Optionnel)

Cliquez sur le bouton 📷 Image

Sélectionnez une image de votre PC

7️⃣

Taper le Commentaire

Tapez votre texte dans l'éditeur

Vous pouvez utiliser gras, italique, listes, etc.

8️⃣

Cliquer "Publier le Commentaire"

Appuyez sur le bouton bleu en bas

Le commentaire s'ajoute immédiatement!

✅ Succès!

Le commentaire apparaît dans la liste des commentaires avec votre emoji et/ou image
