Installer / Configurer Page de Test Aller aux RDV
Prochaines Étapes
  1. Cliquer sur "Installer / Configurer" pour exécuter les migrations SQL
  2. Aller à Rendez-vous dans le menu admin
  3. Cliquer sur le bouton "Voir" (icône oeil) pour un RDV
  4. Tester l'ajout de commentaire avec emoji et image
  5. Vérifier que le commentaire s'affiche correctement
Important
  • Les migrations SQL doivent être exécutées une fois
  • Les commentaires sont stockés en format JSON Delta (compatible Quill.js)
  • Les images sont encodées en base64 (pas de fichier externe)
  • Tous les commentaires sont auto-approuvés (pas de modération)
